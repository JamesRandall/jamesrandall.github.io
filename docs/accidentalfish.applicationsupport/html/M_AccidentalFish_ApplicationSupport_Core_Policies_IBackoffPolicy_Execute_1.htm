Parameters

function
Type: SystemFuncBoolean
The function to run - should return true if it was able to do work, false if not. False will trigger a move to the next stage of the backoff policy. If the function throws an exception the backoff policy will exit and the exception will bubble up to the caller of Execute.
cancellationToken
Type: System.ThreadingCancellationToken
The policy will run until the cancellation token is set
